# Nuii Auto Bulk

> Bulk-automate Adobe Firefly image generation — queue prompts, run large batches unattended, and collect full-resolution results in a single ZIP.

A Chrome extension that automates **bulk image generation on Adobe Firefly**. Load a list of prompts, press Start, and the extension submits them one by one, captures the full-resolution results, and packs them into a ZIP — so you can leave a long batch running unattended.

This is an independent, original project. It contains no third-party extension code, Adobe branding, or proprietary APIs. It drives the normal Firefly web UI using your own signed-in account.

## Features

- **Bulk queue** with pause, resume, stop, retries, and a live run log.
- **Prompt import** by paste, `.txt`, or `.csv`, with automatic duplicate removal.
- **Prefix / suffix transforms**, including `{n}` numbering.
- **Multi-file batches** — drop several prompt files and each one becomes its own segment with its own ZIP.
- **Single-ZIP capture** — results are captured from Firefly's own result data (full resolution) and packed into one archive per run/segment, instead of dozens of individual downloads. Captures are held in IndexedDB so they survive the periodic tab refresh.
- **True 2K on Firefly Image 5** — Firefly's 1K/2K control does not actually change Image 5's output size (it generates 1K either way). This extension forces real 2K by adjusting the generation request, so you get full-resolution images. 1K is also supported.
- **Reliable long runs** — the Firefly tab is refreshed at run start and every 10 prompts to keep result detection accurate once the feed grows large.
- **Self-healing retries** — failed prompts are retried; if Firefly's prompt box fails to render, the tab is reloaded and the prompt resubmitted.
- **Capture reconciliation** — the run summary reports how many images were captured vs. expected and names any prompt whose image did not arrive, so you can re-run just those.
- **Completion chime** when the queue finishes.

## Requirements

- Google Chrome (or a Chromium browser that supports Manifest V3 unpacked extensions).
- An Adobe Firefly account with available generation credits.

## Install (unpacked)

1. Download or clone this repository to a local folder.
2. Open `chrome://extensions`.
3. Enable **Developer mode** (top-right).
4. Click **Load unpacked** and select the project folder.
5. Pin the **Nuii Auto Bulk** icon to your toolbar.

> After editing any extension file, return to `chrome://extensions` and click **Reload** on the extension, then refresh the Firefly tab.

## Usage

1. Open **Adobe Firefly** (`https://firefly.adobe.com/generate/image`) in a tab and sign in.
2. In Firefly, pick your **model** (e.g. Firefly Image 5) and **aspect ratio**.
3. Open the extension popup.
4. Add prompts — paste them (one per line) or import a `.txt` / `.csv` file. Drop multiple files for multi-file batches.
5. Choose your options (resolution, delay, retries, ZIP, etc. — see below).
6. Click **Start** and leave it running.
7. When it finishes, use **Download ZIP** (or the per-file ZIPs saved automatically) to get your images.

## Settings reference

| Setting | What it does |
|---|---|
| **Resolution** | `1K` or `2K`. On Image 5, `2K` is enforced via the generation request. |
| **Delay** | Seconds to wait between prompts. |
| **Timeout** | Max seconds to wait for one generation before giving up. |
| **Retry limit** | How many times to retry a failed prompt. |
| **Continue on error** | Keep going after a failed prompt instead of stopping the run. |
| **ZIP download** | Capture results and pack them into a single ZIP per run/segment. |
| **Dedupe** | Remove duplicate prompts before running. |
| **Prefix / Suffix** | Text added to every prompt (`{n}` inserts the prompt number). |
| **Auto-delete** | Remove generations from Firefly after capture. |
| **Sound on complete** | Play a chime when the queue finishes. |

## Troubleshooting

- **Some images missing from the ZIP** — on very long, laggy runs a result can arrive after its capture window. The run summary names the affected prompts; re-run just those. Generated images also remain in your Firefly history when *Auto-delete* is off.
- **"Prompt input not found"** — Firefly's UI was slow to load; the extension now reloads and retries automatically. If it persists, run fewer prompts per batch.
- **Resolution looks wrong** — confirm the model is Firefly Image 5 and reload the extension after updating.
- **Nothing happens on Start** — make sure a Firefly tab is open and signed in, then reload the extension.

## Privacy

The extension runs entirely in your browser. Prompts, settings, and captured images are stored **locally** (extension storage and IndexedDB). Nothing is sent to any server other than Adobe Firefly itself, as part of normal generation. There is no analytics or telemetry.
